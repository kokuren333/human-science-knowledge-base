# 構成概念妥当性とは何か

## 要点

- 構成概念妥当性とは、知能、不安、自己効力感、衝動性のような直接観察できない心理特性について、得点から行う解釈が理論と証拠によってどれだけ支持されるかを問う考え方である。
- 妥当性は「テストそのものの性質」ではなく、「特定の目的で、その得点をどう解釈し使うか」に関する証拠の総体である[3][4]。
- 1つの相関、1回の因子分析、1つの信頼性係数だけでは十分ではない。内容、反応過程、内的構造、他の変数との関係、利用結果などの証拠を組み合わせる[4]。
- 構成概念妥当性は、一度「確定」するものではなく、理論、測定、データ、解釈を更新し続ける検証プロセスである[1][7]。

![構成概念妥当性の概念地図](../../asset/generated_infographics/構成概念妥当性とは何か/構成概念妥当性とは何か_01.png)

## この記事で答える問い

この記事では、次の問いに答える。

- なぜ心理学では「見えない特性」を測る必要があるのか。
- 構成概念妥当性は、信頼性、基準関連妥当性、内容妥当性とどう違うのか。
- 尺度や検査の得点を、理論的な心理特性の証拠として扱うには何を確認すべきか。
- 研究や臨床で、妥当性を過大評価しないためには何に注意すべきか。

## まず結論

構成概念妥当性は、「この得点は本当にその心理特性を意味しているのか」を、理論とデータの両側から点検する枠組みである。たとえば、ある質問紙が「不安」を測ると主張するなら、単に項目が不安らしく見えるだけでは足りない。項目内容が理論に合っているか、回答者が項目を想定どおりに理解しているか、得点の因子構造が仮説に沿うか、関連する尺度や行動指標と予測どおりに関係するか、そしてその得点を使った判断が妥当な影響をもつかを調べる必要がある[3][4]。

この考え方は、[[認知機能検査は何を測っているのか]]、[[自己概念とは何か]]、[[メタ認知とは何か]]のように、観察しにくい心理過程や自己理解を扱う領域で特に重要になる。

## 背景

心理学の多くの対象は、身長や血圧のように直接測定できない。知能、抑うつ、不安、ワーキングメモリ、共感、自己効力感、衝動性などは、行動、発話、質問紙回答、課題成績、生理指標から推論される。これらの理論上の対象を「構成概念」と呼ぶ。

Cronbach と Meehl は、心理テストの妥当性を、単なる外的基準との一致ではなく、構成概念を含む理論ネットワークの中で検証すべきだと論じた[1]。ここで重要なのは、検査得点が理論の一部として扱われる点である。ある尺度が不安を測るなら、不安がどのような原因、症状、行動、他の特性と関係するかについての仮説があり、その仮説がデータによって支持される必要がある。

その後、Campbell と Fiske は、多特性多方法行列によって、同じ特性を異なる方法で測ったときには相関が高く、異なる特性を同じ方法で測ったときには過度に相関しないことを確認する発想を示した[2]。これは、収束的妥当性と弁別的妥当性の古典的な出発点である。

## 基本概念

### 構成概念

構成概念とは、直接観察できないが、理論上必要とされ、観察可能な指標から推論される概念である。たとえば「[[実行機能とは何か]]」は、課題切り替え、抑制、更新、計画行動などの観察指標から推論されるが、それ自体を直接見ることはできない。

構成概念を測るときには、少なくとも次の3つを分けて考える必要がある。

| 層 | 例 | 問い |
|---|---|---|
| 理論上の特性 | 不安、抑うつ、自己効力感、実行機能 | その特性は何を意味するか |
| 観察指標 | 質問項目、反応時間、正答率、面接評定 | 何を見れば特性の手がかりになるか |
| 得点解釈 | 高得点なら何を言えるか | その得点からどこまで推論してよいか |

構成概念妥当性は、この3層のつながりがどれだけ説得的かを検証する。

### 妥当性

現代的な測定基準では、妥当性は「テストが妥当かどうか」という物の属性ではなく、「特定の利用目的における得点解釈が、証拠と理論によって支持される程度」と考える[4]。同じ尺度でも、集団研究の共変量として使う場合、個人の選抜に使う場合、臨床スクリーニングに使う場合では、必要な妥当性証拠が異なる。

したがって、妥当性の問いは「この尺度は妥当ですか」ではなく、「誰に対して、何のために、どのような判断に使う得点解釈が、どの証拠によって支持されているのか」である。Kane の妥当性論も、得点の解釈と利用について、主張、根拠、反証可能性を含む妥当性論証として組み立てることを重視している[6]。

### 信頼性との違い

信頼性は、測定がどれだけ安定し、一貫しているかに関わる。内的一貫性、再検査信頼性、評定者間一致などが代表例である。しかし、安定して間違う測定もありうる。たとえば、項目同士がよく似ていて高い内的一貫性を示しても、その項目群が理論上の構成概念を狭く、または別物として測っているなら、構成概念妥当性は不十分である。

## 仕組み

構成概念妥当性の検証は、次の循環として理解できる。

![構成概念妥当性の検証循環](../../asset/generated_infographics/構成概念妥当性とは何か/構成概念妥当性とは何か_02.png)

### 1. 構成概念を定義する

最初に、その構成概念が何を含み、何を含まないかを定義する。たとえば「社会不安」を測る尺度であれば、単なる内向性、抑うつ、一般的な緊張、対人場面の回避とどのように異なるのかを明確にする必要がある。

定義が曖昧なまま項目を作ると、後から因子分析をしても、何が見つかったのかを理論的に解釈しにくい。尺度開発では、項目作成の前に構成概念の境界を整理し、必要なら専門家レビューや対象者へのインタビューで内容の妥当性を確認する[7]。

### 2. 観察指標を設計する

次に、構成概念がどのような回答、行動、課題成績として現れるかを考える。質問紙なら項目文、選択肢、想定される回答過程が重要になる。認知課題なら、正答率や反応時間が本当に目的の過程を反映しているかを考える。

ここでは、測定方法そのものが得点に影響する。自己報告では社会的望ましさや自己洞察の限界が、反応時間課題では運動速度や注意状態が混入する。したがって、構成概念妥当性は、測定方法の副作用を切り分ける作業でもある。

### 3. データで検証する

データ分析では、複数の証拠を組み合わせる。

| 証拠の種類 | 典型的な方法 | 見たいこと |
|---|---|---|
| 内容に基づく証拠 | 専門家レビュー、項目マッピング | 項目が構成概念の範囲を十分に覆うか |
| 反応過程に基づく証拠 | 認知面接、回答プロセス分析 | 回答者が想定どおりに項目を理解しているか |
| 内的構造に基づく証拠 | 探索的因子分析、確認的因子分析、項目反応理論 | 得点構造が理論と合うか |
| 他の変数との関係 | 収束・弁別・予測・既知集団差 | 関連概念と予測どおり関係するか |
| 利用結果に関する証拠 | 判断結果、介入影響、公平性の検討 | 得点利用が意図した目的に照らして妥当か |

この分類は、2014年版 *Standards for Educational and Psychological Testing* の妥当性証拠の整理と対応している[4]。

### 4. 解釈を更新する

データが理論と合わない場合、単に「尺度が失敗した」とは限らない。理論定義が広すぎた、項目が一部の下位側面に偏った、回答者集団によって意味が変わった、方法効果が強かった、など複数の可能性がある。構成概念妥当性の検証は、得点を守る作業ではなく、得点解釈をより正確にする作業である。

Borsboom らは、妥当性をめぐる議論で、測定対象となる属性の存在と、その属性が測定結果に因果的に影響する過程を重視した[5]。これは、相関パターンだけではなく、「なぜその得点がその特性を反映すると言えるのか」という測定メカニズムを問う視点である。

## 図解

構成概念妥当性では、証拠を1種類に絞らない。特に重要なのは、内的構造、外的関係、反応過程と利用結果を組み合わせることである。

![妥当性証拠の組み合わせ](../../asset/generated_infographics/構成概念妥当性とは何か/構成概念妥当性とは何か_03.png)

実務上は、次のような流れで検討すると整理しやすい。

1. 何を測るのかを理論的に定義する。
2. その特性が観察指標に現れる理由を説明する。
3. 予測される相関、群間差、因子構造、発達変化、介入効果を事前に置く。
4. データが予測に合うかを確認する。
5. 合わない場合、理論、項目、対象集団、方法効果、利用目的のどこを修正すべきかを考える。

## 臨床・研究との接続

研究では、構成概念妥当性が弱いと、変数名だけが理論的で、実際には別のものを分析している危険がある。たとえば「[[認知的柔軟性とは何か]]」を測る課題が、実際には処理速度や課題理解の差を強く反映している場合、その得点を認知的柔軟性として解釈するには追加の証拠が必要になる。

臨床では、尺度得点を個人の状態理解やスクリーニングに使うことがある。ただし、尺度得点は診断そのものではない。教育・研究目的で言えば、得点は面接、生活史、観察、他者報告、機能障害、文化的背景と合わせて解釈されるべき情報である。特に精神医学・臨床心理学では、得点の妥当性は集団研究で支持されていても、個人の診断や治療方針にそのまま移せるとは限らない。

尺度開発では、Clark と Watson が述べるように、妥当な測定器を作るには、項目プール作成、内容領域の明確化、項目分析、因子構造、外的基準、交差検証を段階的に進める必要がある[7]。これは「よく相関した項目を集める」だけではなく、「理論的に意味のある得点を作る」作業である。

## よくある誤解

### 誤解1: 高い信頼性があれば妥当である

信頼性は妥当性の必要条件に近いが、十分条件ではない。測定が不安定であれば妥当な解釈は難しいが、測定が安定していても、狙った構成概念を測っているとは限らない。

### 誤解2: 因子分析で1因子が出れば構成概念妥当性は示された

因子分析は内的構造に関する証拠であり、重要ではある。しかし、1因子構造が得られても、その因子が理論上の構成概念を意味するかは別問題である。他尺度との関係、行動指標、反応過程、利用目的に照らした証拠が必要である。

### 誤解3: 妥当性には種類があり、どれか1つを示せばよい

古い整理では、内容妥当性、基準関連妥当性、構成概念妥当性が別々の種類のように扱われることがあった。しかし Messick 以降の統合的見方では、これらは得点解釈を支える証拠の異なる側面として理解される[3]。構成概念妥当性は、それらをまとめる上位の枠組みとして働く。

### 誤解4: 妥当性は尺度開発者だけが気にすればよい

既存尺度を使う研究者も、対象集団、言語、文化、測定場面、利用目的が変わるなら妥当性を再検討する必要がある。同じ尺度でも、成人向け尺度を青年に使う、英語尺度を日本語訳して使う、研究用尺度を臨床判断に使う、といった場合には追加の証拠が求められる。

## 理解チェック

1. 構成概念妥当性が「テストそのもの」ではなく「得点解釈」に関わると言われるのはなぜか。
2. 高い内的一貫性があっても、構成概念妥当性が不十分な例を1つ考えられるか。
3. 収束的妥当性と弁別的妥当性は、それぞれ何を確認する証拠か。
4. ある尺度を別の文化・年齢集団に使うとき、どのような妥当性証拠を追加で集めるべきか。

## 未解決問題

- 心理特性を「潜在変数」として扱うモデルと、症状・行動・認知過程のネットワークとして扱うモデルでは、妥当性の考え方がどのように変わるか。
- 質問紙、行動課題、神経指標、スマートフォンログを統合するとき、どの水準の構成概念を妥当化していると言えるか。
- 臨床で用いる尺度得点について、集団レベルの妥当性証拠を個人レベルの判断にどう接続すべきか。
