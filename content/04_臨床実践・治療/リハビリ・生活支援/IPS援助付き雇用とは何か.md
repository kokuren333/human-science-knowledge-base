# IPS援助付き雇用とは何か

## 要点

- IPS援助付き雇用は、精神疾患やメンタルヘルス上の困難をもつ人が、本人の希望に基づいて一般就労を得て維持するための、エビデンスに基づく就労支援モデルである[1][2]。
- 中核は「訓練してから働く」ではなく、「働く場に早く入り、働きながら支える」という考え方である[2][3]。
- 主要原則には、一般就労への焦点、本人の希望による参加、精神保健サービスとの統合、本人の好みの重視、福利厚生相談、迅速な求職、系統的な職場開発、期限を切らない個別支援が含まれる[2]。
- 研究では、通常支援に比べてIPS群の競争的雇用獲得が高いことが一貫して示されている。一方、生活の質や精神症状など非就労アウトカムの効果は、就労アウトカムほど強くはない[3][4]。
- 日本で実装するには、医療、障害福祉、職業リハビリテーション、雇用制度の分断を前提に、IPS原則とフィデリティを保ちながら地域の制度に合わせる必要がある[7][8]。

## この記事で答える問い

1. IPS援助付き雇用は、通常の就労支援や職業訓練と何が違うのか。
2. なぜ「早く一般就労を目指す」ことが支援として成立するのか。
3. どのような研究根拠があり、どこに限界があるのか。
4. 日本の地域精神医療・障害福祉の文脈では、何に注意して実装すべきか。

## まず結論

IPS援助付き雇用は、就労を治療や生活支援の「後」に置くモデルではない。本人が働きたいと希望するなら、診断名、症状の重さ、過去の失敗、就労準備性だけで排除せず、一般就労を目標に、求職活動を早く始め、就職後も本人・職場・医療生活支援チームをつないで支えるモデルである[1][2]。

ただし、IPSは「とにかく早く就職させる」方法ではない。早さは、本人の希望を軽視するためではなく、長い訓練や選別によって働く機会を失わせないためにある。臨床的には、[[意思決定支援とは何か]]や[[リカバリー志向支援とは何か]]と同じく、本人の価値観、希望、リスク、環境調整を具体化する支援として理解するとよい。

## 背景

精神疾患をもつ人の多くは、働く意欲や能力があっても、症状、偏見、職場調整の不足、医療・福祉・雇用サービスの分断、社会保障への不安によって就労から遠ざけられやすい。従来型の職業リハビリテーションでは、訓練、作業所、準備プログラム、段階的な移行を経てから一般就労へ進む発想が強かった。

IPSはこの順序を組み替える。重い精神障害をもつ人でも、本人が望む仕事に近い実際の職場で、必要な支援を受けながら働くことができるという前提に立つ。Cochraneレビューは、援助付き雇用が競争的雇用の日数や就職までの速さを改善しうるとまとめているが、根拠の質や非就労アウトカムには注意が必要だとしている[4]。

欧州6施設のEQOLISE試験でも、IPS群は通常の職業サービスより競争的雇用に就く割合が高かった。ただし、効果は地域の制度、労働市場、実装の質に左右されるため、単に名称だけを導入しても同じ結果になるとは限らない[5][6]。

## 基本概念

### IPSとは何か

IPSは Individual Placement and Support の略で、日本語では「個別型援助付き雇用」「個別就労支援プログラム」「IPS援助付き雇用」などと訳される。SAMHSAの2025年版ツールキットは、IPSを、精神疾患をもつ人が競争的で意味のある仕事を見つけ、維持するためのエビデンスに基づく援助付き雇用モデルとして整理している[1]。

ここでいう「競争的雇用」または「一般就労」とは、障害者専用に切り出された場だけではなく、地域の通常の労働市場で、他の労働者と同じように応募でき、少なくとも最低賃金以上の賃金が支払われる仕事を指す[2]。

### 8つの原則

IPS Employment Centerは、IPSの8原則を次のように整理している[2]。

| 原則 | 意味 |
|---|---|
| 一般就労への焦点 | ボランティアや保護的就労だけでなく、通常の労働市場での仕事を目標にする |
| 本人の希望による参加 | 症状、診断、過去の失敗、準備性だけで除外しない |
| 医療・生活支援との統合 | 就労支援専門職が精神保健チームと連携する |
| 本人の好みの重視 | 支援者の判断より、本人の希望、強み、関心を出発点にする |
| 福利厚生・社会保障の相談 | 働くことが給付、医療費、生活に与える影響をわかりやすく扱う |
| 迅速な求職 | 長い事前訓練より、早期に職場との接点をつくる |
| 系統的な職場開発 | 本人の希望に沿って雇用主との関係を継続的に開発する |
| 期限を切らない個別支援 | 就職後も本人が必要とする限り支援を調整する |

この原則の組み合わせが重要である。たとえば「迅速な求職」だけを取り出すと、急かす支援になりうる。しかし「本人の好み」「社会保障相談」「医療との統合」「継続支援」と結びつくことで、働く場に出た後も調整し続けるモデルになる。

## 仕組み

IPSの仕組みは、単純な職業紹介ではなく、本人、職場、医療・生活支援、制度相談を同時に動かす点にある。

1. 本人が「働きたい」「学びたい」「収入を得たい」「社会とつながりたい」といった希望を表明する。
2. 就労支援専門職は、症状の完全な安定や長期訓練を待たず、希望に近い仕事を探し始める。
3. 求職中も、[[ケースマネジメントとは何か]]、医療チーム、福祉制度、家族やピアサポートと連携する。
4. 就職後は、勤務時間、仕事内容、対人関係、通院、服薬、疲労、職場への開示範囲などを継続的に調整する。
5. うまくいかなかった場合も「失敗」とみなさず、希望と環境を見直して次の仕事や学びへつなぐ。

![従来型支援とIPSの違い](../../asset/generated_infographics/IPS援助付き雇用とは何か/IPS援助付き雇用とは何か_02.png)

「働きながら支える」とは、本人を準備不足のまま放り出すことではない。むしろ、実際の職場で明らかになる困りごとを、本人の経験に即して扱うということである。これは[[精神科リハビリテーションとは何か]]の一部であり、[[就労支援とは何か]]を医療・生活支援と切り離さないための実装モデルでもある。

## 図解

IPSを臨床現場で考えるときは、次の3層に分けると整理しやすい。

| 層 | 見るポイント | 典型的な問い |
|---|---|---|
| 本人の希望 | どんな仕事、働き方、収入、意味を望むか | 「何を大切にして働きたいか」 |
| 支援プロセス | 求職、職場開発、福利厚生相談、継続支援が動いているか | 「待たせすぎていないか」「支援が途切れていないか」 |
| 実装の質 | IPS原則に沿っているか、チーム統合があるか | 「名前だけIPSになっていないか」 |

## 臨床・研究との接続

### 臨床との接続

IPSは、[[精神疾患と就労困難はどう関係するのか]]という問いに対して、「働けるかどうか」を個人内の能力だけで判断しない。症状、認知機能、対人不安、生活リズム、通勤、職場文化、制度上の不安が絡み合っているため、支援も多層的である。

臨床家にとって重要なのは、就労を「症状がよくなった後の報酬」としてではなく、回復を支える生活上の役割として扱うことである。仕事はストレス源にもなりうるが、収入、所属、日課、自己効力感、将来展望をもたらすこともある。したがって、医療者はリスクを評価しつつ、本人の選択を過度に制限しない姿勢が求められる。

### 研究との接続

RCTのメタ解析では、IPSは通常支援よりも競争的雇用獲得、就労期間、収入などの就労アウトカムで優れていた[3]。Cochraneレビューも、重い精神疾患をもつ成人に対して援助付き雇用が就労アウトカムを改善しうるとまとめている[4]。一方で、生活の質、精神症状、入院、費用対効果などは研究間でばらつきがあり、「就労が増えればすべての臨床アウトカムが自動的に改善する」とは言えない[3][4]。

この点は、研究評価で重要である。IPSの主要アウトカムは就労率だけでは足りない。本人にとって意味のある仕事か、働き続けることで生活の質がどう変わるか、症状悪化時に支援がどう働くか、制度上の不利益をどう防ぐかも評価対象にする必要がある。

### 日本での実装

日本では、医療機関、障害福祉サービス、ハローワーク、障害者職業センター、企業、自治体が別々の制度で動きやすい。そのため、IPSの「統合されたサービス」をそのまま輸入するだけでは実装が難しい。国立精神・神経医療研究センターでは、日本の制度下でIPS原則を大切にした個別就労支援を評価するため、日本版個別型援助付き雇用フィデリティ尺度の開発・検証が進められてきた[7]。

また、日本語文献では、IPSがディーセント・ワークや地域ケアと接続しうる一方、利用可能性、制度化、医療と雇用の連携、支援者養成が課題として論じられている[8]。したがって、日本でのIPSは「海外で有効だった技法」ではなく、地域の制度と労働市場に合わせて、原則とフィデリティを保つ実装課題として見る必要がある。

## よくある誤解

### 誤解1: 症状が安定してからでないと働けない

安全配慮は必要だが、「完全に安定してから働く」という条件を置くと、多くの人が機会を失う。IPSは、症状が残っていても、本人の希望とリスクを具体的に見ながら、現実の仕事に近い場で支援を組み立てる。

### 誤解2: IPSは就職件数を増やすだけの制度である

IPSの目的は、支援機関の就職実績を増やすことではない。本人が望む仕事を見つけ、必要な限り支援し、働くことが生活や回復にどう位置づくかを一緒に調整することである。

### 誤解3: 迅速な求職は本人を急かすことだ

迅速な求職は、本人の意思を飛ばすことではない。長い選別や訓練で機会を先送りせず、本人が希望する仕事との接点を早くつくるという意味である。ここでは[[動機づけ面接とは何か]]や[[自己決定理論とは何か]]とも接続し、本人の自律性を支える対話が重要になる。

### 誤解4: IPSは医療と関係のない雇用支援である

IPSでは、就労支援専門職が精神保健チームと連携する。通院、薬物療法、睡眠、疲労、対人ストレス、危機時対応、家族支援などは、仕事の継続に直接関係する。これは[[多職種連携は地域精神医療でなぜ重要なのか]]の実践例でもある。
