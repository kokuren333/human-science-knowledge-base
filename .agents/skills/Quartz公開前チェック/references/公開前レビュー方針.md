# 公開前レビュー方針

公開前レビューでは、技術的互換性だけでなく、読者への安全性と誤読リスクを見る。

## 医療・精神医学

- 個別診断や治療指示として読める断定を避ける。
- 緊急リスクに関わる話題では相談先や救急につながる注意を置く。
- 確立知見、臨床慣行、仮説を分ける。

## 研究・数理モデル

- モデルの仮定と限界が書かれている。
- 式の記号が定義されている。
- 実装例がある場合、環境や依存関係が明確である。

## 公開判断

ユーザーが明示した場合だけ `draft: false`, `publish: true` を提案する。未検証文献や個人メモが残る場合は draft のままにする。
