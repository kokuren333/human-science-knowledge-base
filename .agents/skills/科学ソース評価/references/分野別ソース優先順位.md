# 分野別ソース優先順位

## 医学・精神医学

診療ガイドライン、DSM / ICD、システマティックレビュー、メタ解析、大規模 RCT、コホート研究、主要レビュー、古典的原著、教科書の順に見る。

## 神経科学・認知科学

レビュー誌、主要教科書、重要原著、方法論的批判、学会・大学・研究機関の解説を組み合わせる。

## 機械学習・数理モデル

原著論文、公式ドキュメント、標準教科書、サーベイ、講義資料、実装リポジトリを使う。実装上の主張は公式 API やバージョンを確認する。

## 哲学・倫理・社会

査読論文、専門書、主要機関の報告書、政策文書を優先する。価値判断と経験的主張を分ける。
