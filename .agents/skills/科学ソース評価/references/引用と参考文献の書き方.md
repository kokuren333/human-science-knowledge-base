# 引用と参考文献の書き方

## 基本

参考文献には、著者、年、タイトル、媒体、DOI / PMID / arXiv ID / URL のいずれかを含める。文献が実在するか確認できない場合は、候補として扱い、確定文献として書かない。

## 本文での使い方

本文では読みやすさを優先し、重要な主張にだけ参照番号や文献名を添える。長い逐語引用は避け、要約と言い換えで示す。

## 書式例

```markdown
- Friston K. The free-energy principle: a unified brain theory? Nature Reviews Neuroscience. 2010. DOI: 10.1038/nrn2787. URL: https://doi.org/10.1038/nrn2787
```

## 見解が割れる場合

単一の結論に寄せず、争点、方法論の違い、対象集団、測定方法を明示する。
