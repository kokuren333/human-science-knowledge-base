# AGENTS.md

このリポジトリは、Obsidian / Quartz 向けの人間科学ナレッジベースを構築するための作業場所です。

## 基本方針

- ノート本文は原則として日本語で書く。
- 脳・神経・認知・精神医学・心理学・機械学習・数理モデル・計算論的精神医学を中心に扱う。
- ただし他分野の依頼でも、適切なディレクトリ、MOC、タグ、関連ノート候補を設定する。
- 既存ノートを編集する前に、関連ファイルを読み、文脈を把握する。
- 無断で大量のファイル移動・リネーム・削除をしない。
- 医療・精神医学に関する内容は教育・研究目的として記述し、個別診断や治療指示として断定しない。

## Obsidian / Quartz 方針

- 内部リンクは `[[ノート名]]` 形式を使う。
- Quartz 公開を意識し、frontmatter を整える。
- 長文ノートでは `description`, `tags`, `aliases`, `draft`, `publish`, `created`, `updated`, `enableToc` を原則として含める。
- 数式は Markdown / Quartz で読める LaTeX を使う。

## 完了条件

- 新規ノート作成時は、関連ノート候補、MOC 更新候補、参考文献、未解決問題を含める。
- スクリプトを変更した場合は、可能な範囲でテストまたは検証コマンドを実行する。
- 変更後は、作成・更新したファイルと主な意図を要約する。
